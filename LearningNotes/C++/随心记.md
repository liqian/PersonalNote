## operation () 重载
1. 可以实现可隐式转化
```C++
class A
{
    enum EnumType
    {
        ENUM_ONE=1,
        ENUM_TWO=2
    };

    A(){};
    ~A(){};

    operator EnumType() const{ return m_value;}//这里实现了类A 到枚举EnumType的隐式转化
    
private:
    EnumType m_value;
}

 A a;
 if( A::ENUM_ONE != a){}//如果没有重载 operator EnumType() 这里会出现编译失败，应为类型不一致，不能进行比较
```
